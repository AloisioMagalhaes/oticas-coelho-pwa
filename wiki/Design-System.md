## Paleta de Cores

### Primary (Azul Confiança)
Transmite confiança, profissionalismo e saúde visual.

| Token | Hex | Uso |
|-------|-----|-----|
| --color-primary-500 | #1B6FF0 | Primary base |
| --color-primary-600 | #1559C2 | Hover states |
| --color-primary-700 | #0F4496 | Active states |

### Secondary (Dourado Elegância)
Agrega valor percebido, remetendo a elegância acessível.

| Token | Hex | Uso |
|-------|-----|-----|
| --color-secondary-500 | #F5A623 | Secondary base |
| --color-secondary-400 | #FFD54D | Highlights |

### Neutros
Escala de cinza completa para textos, backgrounds e bordas.

## Tipografia

| Token | Fonte | Uso |
|-------|-------|-----|
| --font-family-heading | Poppins | Títulos, CTAs |
| --font-family-body | Inter | Corpo de texto |
| --font-family-accent | Playfair Display | Destaques italic |

## Espaçamento

Grid base de **8px**. Todos os espaçamentos são múltiplos de 8.

## Componentes

Seguindo **Atomic Design**:
- **Átomos**: Button, Input, Badge, Icon, Card, Avatar
- **Moléculas**: SearchBar, ProductCard, NavItem, FormField
- **Organismos**: Header, Footer, HeroBanner, ProductGrid, Testimonials
- **Templates**: MainLayout
- **Páginas**: Home, Produtos, Sobre, Contato
