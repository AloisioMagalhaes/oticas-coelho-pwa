## Stack

| Camada | Tecnologia |
|--------|------------|
| Components | Lit (Web Components nativos) |
| Bundler | Vite 5 |
| Language | TypeScript 5.3 |
| Styling | CSS Custom Properties (Design Tokens) |
| PWA | Vite PWA Plugin + Workbox |
| Docs | Storybook 7 |
| CI/CD | GitHub Actions |
| Hosting | GitHub Pages |
